Sprint 3 - Cylinder Detectiion

Summary of Executable Code

The code will do the following:

  1. spawn_cylinder.cpp will summon in the cylinder at a specified location.
  2. detect_cylinder.cpp will attempt to detect the cylinder.

Detection Method

The executable will first read in laser scan values and will filter out 'bad' readings. The next stage invovles detecting which have the potential to be a cylinder. The final stage goes through a vector of detected objects and determines the best match.
